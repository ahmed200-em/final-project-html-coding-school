# 📄 Array Page – Full Website (HTML, CSS, JS)

## 🚀 Project Overview

**Array Page** is a modern multi-page website built using **HTML, CSS, and JavaScript**.  
It was created as a team project to practice frontend development, UI structuring, and responsive design.

The project includes multiple pages such as Home, Blog, Services, and Signup, all designed with a clean and consistent interface.
